# vantajs-365

Vibe Coding 練習 Vanta.js - 一個基於 Three.js 的 3D 背景效果互動演示應用，使用 Gemini Canvas 製作。

## 🎨 概述

這是一個利用 **Vanta.js** 建立的即時互動 3D 背景效果展示平台。提供三種不同的動畫效果，使用者可以透過右側設定面板即時調整各項參數，實時預覽效果變化。

## ✨ 主要特性

- **三種 3D 背景效果**
  - **Net (科技網狀)**：由動態點線組成的科技感網狀結構
  - **Birds (群鳥飛舞)**：逼真的鳥群飛行動畫
  - **Waves (流動波浪)**：光滑流動的波浪效果

- **即時互動控制面板**
  - 支援即時色彩調整（顏色與速度即時生效）
  - 支援重量參數調整（數量與大小放開後生效）
  - 響應式設計，支援平板和行動裝置

- **使用者友善的界面**
  - 摺疊式設定面板，不影響視覺體驗
  - 數值實時顯示標籤
  - 美觀的玻璃態設計 (Glassmorphism)
  - 自定義滾動條樣式

## 🚀 快速開始

### 使用方法

1. **打開應用**
   - 在瀏覽器中打開 `index.html` 檔案

2. **選擇效果**
   - 點擊右上角的設定按鈕 ⚙️ 打開設定面板
   - 從「選擇效果」下拉選單選擇所需的效果

3. **調整參數**
   - 使用顏色選擇器調整色彩（即時生效）
   - 使用滑塊調整其他參數
   - 對於標記 ⚡ 的參數，放開滑塊後生效

4. **關閉面板**
   - 點擊「✕」按鈕或外部區域關閉設定面板

## 💡 核心 JavaScript 邏輯

### 效果配置系統
應用使用統一的配置對象管理三種效果，每個效果定義：
- 預設參數
- 控制項設定（含是否需重繪標記）

### 參數更新機制
- **即時更新（Instant）**：顏色、速度等輕量參數直接調用 `setOptions()`
- **重繪更新（Reload）**：數量、大小等結構參數需銷毀後重新初始化

### 互動流程
1. 使用者在 UI 調整參數
2. 系統判斷參數類型和重繪需求
3. 根據需求執行即時更新或重繪操作
4. 視覺反饋實時顯示變化

## 🎮 互動提示

| 操作 | 效果 |
|------|------|

## 📱 響應式設計

- 支援桌面電腦、平板和行動裝置
- 自適應的文字大小和排版
- 觸控友善的控制項
- 全屏 3D 背景效果

## 🔧 自定義修改

若要修改效果參數或外觀，可編輯 `effectsConfig` 物件中的對應項：

```javascript
'net': {
    defaults: {
        // 修改預設參數
        points: 12
    },
    controls: [
        // 新增或修改控制項
        { id: 'newParam', label: '新參數', type: 'range', min: 0, max: 100, step: 1, default: 50, reload: false }
    ]
}
```
